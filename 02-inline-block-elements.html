Elementos em bloco versus elementos inline

HTML

<em>primeiro</em>
<em>segundo</em>
<em>terceiro</em>

<p>quarto</p>
<p>quinto</p>
<p>sexto</p>

Saída:

primeiro segundo terceiro

quarto

quinto

sexto

O elemento <em> é inline, então como você pode ver ao lado, os três primeiros elementos ficam na mesma linha uns dos outros sem espaço entre eles.

O <p>, por outro lado, é um elemento em bloco, então cada elemento aparece em uma nova linha, com espaço acima e abaixo de cada um (o espaçamento é devido à estilização CSS padrão que o browser aplica aos parágrafos).
